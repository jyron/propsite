# propsite
A site for people to submit detailed proposals to solve local or national issues.  
# to run it just 
    make a virtual environment in root folder 'python3 -m venv venv'
    enter it 'source venv/bin/activate'
    install requirements 'pip install -r requrements.txt'
    export app and debug environment 'export FLASK_APP=app' and 'export FLASK_ENV=development'
    run it!  'flask run'
